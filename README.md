Title  
Number Guessing Game

Description  
A simple Python terminal game where the player guesses a number between 1 and 100.

Features
Random number generation
User input with hints (too high / too low)
Tracks number of attempts

Usage  
Just say: Run the script with Python and follow the prompts in the terminal.
